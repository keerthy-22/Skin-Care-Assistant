# 🌿 Skin Care Recommendation System

A Python-based skincare recommendation application that helps users identify their skin type and receive personalized skincare guidance based on their skin concerns.

## 📖 Project Overview

The Skin Care Recommendation System is designed to provide basic skincare recommendations using a structured knowledge base. Users first learn how to identify their skin type using a simple 30-minute skin test, then select their skin type and the concern they are experiencing. Based on the selection, the application displays:

* A brief explanation of the selected skin concern
* Recommended skincare ingredients
* Suitable product categories
* Morning skincare routine
* Night skincare routine
* Helpful skincare tips

The project is built using **Python** and **Streamlit**, making it simple, interactive, and beginner-friendly.

## ✨ Features

* 🧴 Skin type identification guide
* 👤 Supports five skin types:

  * Dry Skin
  * Oily Skin
  * Combination Skin
  * Normal Skin
  * Sensitive Skin
* 📋 Covers seven common concerns for each skin type
* 🧪 Ingredient recommendations for every concern
* 🛍️ Suggested product categories
* 🌞 Personalized morning skincare routine
* 🌙 Personalized night skincare routine
* 💡 Additional skincare tips
* 🎨 Interactive user interface built with Streamlit
